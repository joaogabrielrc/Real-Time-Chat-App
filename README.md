<h1 align="center">Real Time Chat App</h1>

## Common setup

Clone the repository.

```bash
$ git clone git@github.com:joaogabrielrc/Real-Time-Chat-App.git
$ cd Real-Time-Chat-App
```

## Running the server

```bash
$ cd server
$ npm install
```

```bash
npm run start:dev
```

## Running the client

```bash
$ cd client
$ npm install
```

```bash
npm run dev
```

## Test

```bash
# unit tests
$ npm run test
```
